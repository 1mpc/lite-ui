# 介绍

Vue3-LiteUI 是一个**轻量、现代、智能**的 Vue 3 组件库，专为 2025-2026 年的前端开发趋势而设计。

## 设计理念

```
轻量 | 智能 | 无障碍 | 现代化
```

## 核心特性

### 🎨 智能主题系统

- 自动检测系统主题偏好
- 支持亮色/暗色模式无缝切换
- CSS 变量驱动的主题定制

### 🤖 AI 输入组件

- 智能提示和自动补全
- 支持异步数据源
- Tab 键快速接受建议

### ✨ 微交互动画

- 精心设计的涟漪效果
- 平滑过渡动画
- 支持 prefers-reduced-motion

### ♿ 无障碍优先

- WCAG 2.1 AA 级标准
- 完整的键盘导航
- 屏幕阅读器支持

### ⌨️ 命令面板

- 类似 VS Code 的命令面板
- 快捷键唤起 (Ctrl+K)
- 模糊搜索

### 📦 极致轻量

- 核心包体积 < 50KB gzip
- 零第三方依赖
- Tree-shaking 友好

## 技术栈

- Vue 3.4+
- TypeScript 5.0+
- Vite 5.0+
- CSS Variables
